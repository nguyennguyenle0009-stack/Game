## Tính năng

- Di chuyển nhân vật 2D
	
## Cách chạy

1. Mở project trong Eclipse
2. Chạy class `Main.java` bằng cách click chuột phải → Run As → Java Application

## Yêu cầu hệ thống

- Java SE 17 trở lên
